Informações do projeto

GoGoType

GOGOTYPE é um projeto de uma plataforma de jogo que usa como base a digitação para praticar as habilidades do jogador. O conceito original do projeto é de desafiar a capacidade do jogador com suas habilidades de digitação como parte da experiência do jogo. Atualmente o projeto está em desenvolvimento com a disponibilidade de uma ferramenta simples de digitação. Um protótipo onde o usuário tem que digitar corretamente a frase apresentada na tela. Com essa ferramenta o usuário pode treinar sua digitação a partir de frases aleatórias que vão surgindo para que o jogador digite de forma correta e melhore suas habilidades de digitação. Com essa ferramenta simples é possível que o usuário aprimore sua cognição para assemelhar o que está sendo mostrado em texto com o que está sendo digitado pelo usuário.

GOGOTYPE está começando como um projeto simples, como uma ferramenta de treinamento de digitação com alguns desafios para o jogador. Em futuras atualizações e aprimoramento da plataforma visamos em adicionar elementos para melhorar a experiência na plataforma para os usuários. Os principais elementos a adicionar são efeitos visuais, aprimoramento e adição de funções, modos de jogo e efeitos sonoros. Visamos em crescer este projeto em algo além de uma simples ferramenta para digitar. Como está ainda no inicio do desenvolvimento estamos testando novas formas, modos e as ferramentas tentando implementa-las no projeto. Com isso o projeto pode acabar se transformando em algo maior.

Em atualizações futuras pretendemos adicionar:


Modos de jogo

Um modo desafio que apresentará um número x de diversas frases para que o jogador escreva de forma correta. À medida que o jogador vai digitando de maneira correta irá prosseguindo para outras frases até completar o desafio. Cada desafio terá uma quantidade x de frases apresentadas e níveis de dificuldade.

Nível de dificuldade

>Adicionar níveis de dificuldades com desafios para que os usuários possam treinar suas habilidades de digitação na dificuldade escolhida. Cada dificuldade terá seus próprios desafios que os usuários podem estar treinando para aprimorar seus tempos e erros ao tentar completar os desafios. Os desafios contam com uma margem de erros de digitação, tempo para terminar o desafio baseados no nível da dificuldade escolhida. Quanto maior a dificuldade menos erros serão tolerados e o tempo será reduzido.

Efeitos visuais.

em atualizações futuras adicionar efeitos gráficos para melhorar aspectos do projeto melhorando a interação de ações dos usuários e aumentando a qualidade do projeto para uma experiência satisfatória.

Adição de modelos de sprites para cada jogador com animações de ataque quando o desafio é completo. Quando o usuário completar um desafio primeiro que seu oponente ou se ganhar uma disputa de desafios haverá animações de personagens para cada ação completa por um usuário como forma de indicativo de informações.

Efeitos sonoros.

Adição de efeitos sonoros melhorando a qualidade do projeto e que incremente melhor a experiência do jogador. A implementação dos efeitos sonoros vai deixar uma experiência mais agradável com efeitos sonoros que indicam que o jogador digitou corretamente as palavras ou que está errado, indicar que o desafio foi concluído de forma correta, músicas enquanto o jogo está sendo executado e outros detalhes

Sistema de pontuação

À medida que os jogadores vão completando os desafios vai gerando uma pontuação para cada ação feita durante o jogo. Ou seja, a cada desafio completo corretamente o usuário ganha um número x de pontuações. Essas pontuações tem como base também o tempo que o jogador leva para escrever cada frase apresentada na tela.
