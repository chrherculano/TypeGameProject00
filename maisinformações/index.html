GOGOTYPE é um projeto de uma plataforma de jogo que usa como base a digitação para praticar as habilidades do jogador. O conceito original do projeto é de desafiar a capacidade do jogador com suas habilidades de digitação como parte da experiência do jogo. Atualmente o projeto está em desenvolvimento com a disponibilidade de uma ferramenta simples de digitação. Um protótipo onde o usuário tem que digitar corretamente a frase apresentada na tela. Com essa ferramenta o usuário pode treinar sua digitação a partir de frases aleatórias que vão surgindo para que o jogador digite de forma correta e melhore suas habilidades de digitação. Com essa ferramenta simples é possível que o usuário aprimore sua cognição para assemelhar o que está sendo mostrado em texto com o que está sendo digitado pelo usuário. GOGOTYPE está começando como um projeto simples, como uma ferramenta de treinamento de digitação com alguns desafios para o jogador. Em futuras atualizações e aprimoramento da plataforma visamos em adicionar elementos para melhorar a experiência na plataforma para os usuários. Os principais elementos a adicionar são efeitos visuais, aprimoramento e adição de funções, modos de jogo e efeitos sonoros. Visamos em crescer este projeto em algo além de uma simples ferramenta para digitar. Como está ainda no inicio do desenvolvimento estamos testando novas formas, modos e as ferramentas tentando implementa-las no projeto. Com isso o projeto pode acabar se transformando em algo maior.
